## 提示
项目实用create-react-app创建，非常适合新手学习，也是我学习用来理解react+redux编写。

## 技术栈
react@16.4.1 + redux@4.0.0 + react-router-dom@4.3.1 + webpack@3.8.1 + less@3.5.3 + antd@3.6.5

## 运行项目
```
git clone https://github.com/tianjie23/bookmanager.git

 cd react-book

 npm i  或者运行  yarn(推荐)
  
 npm start

 npm run build （发布）
```
